# DMD — Digital Marketing Dashboard

A comprehensive, full-funnel digital marketing dashboard built with **React + Vite**, designed for clients to track progress across all digital channels at a glance.

![Dashboard Screenshot](https://github.com/user-attachments/assets/069c637a-29fc-4a0a-9a7b-a33e4cc21caf)

## Features

| Tab | What you get |
|-----|-------------|
| **Overview** | 8 top-level KPI cards, 6-month growth trend chart, NPS breakdown, Wix traffic sources, AZ market share, UX video-tracing retention, content velocity, Google review pipeline |
| **Social** | Per-platform reach / engagement / follower cards, Social Intelligence composed chart, weekly engagement grouped bar chart |
| **SEO & Content** | Organic growth KPIs, keyword ranking table with position changes, top blog performance list, TikTok video velocity |
| **Paid Ads** | Channel performance table (Google / Meta / LinkedIn), blended CPL, 6-month lead volume trend chart |
| **Email** | Campaign-level open, click, and conversion metrics, grouped bar chart |
| **Pipeline** | Priority-bucketed action items with due dates, Google review outreach tracker |

## Tech Stack

- [React 19](https://react.dev/)
- [Vite 7](https://vite.dev/)
- [Recharts](https://recharts.org/) — charts & visualisations
- [Lucide React](https://lucide.dev/) — icons
- [Tailwind CSS v4](https://tailwindcss.com/)

## Getting Started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
